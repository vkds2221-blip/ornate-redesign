## 🎨 Design Decisions

- **Dark theme** — Positions Ornate in the premium IT services tier, consistent with competitors like Nagarro, Persistent Systems, and Mphasis
- **Typography** — Syne (display/headings) + DM Sans (body) for editorial authority with readability
- **Hero section** — Clear conversion hierarchy: badge → headline → sub-copy → dual CTA → stat strip
- **Services grid** — 1px gap pattern for a modern, high-end feel
- **Stats section** — Large typographic numbers used as trust anchors
- **Certifications** — ISO, CMMI Level 3, and Microsoft credentials surfaced as structured badge components
- **Marquee ticker** — Keeps services visible between sections without cluttering the layout
- **Single-file delivery** — All CSS and JS embedded inline; no external dependencies except Google Fonts
